# <a name="card-schema"></a>Схема карты

Карты
   * [`AdaptiveCard`](#schema-adaptivecard) — Корневой элемент в инструмент Adaptive cards.

Элементы карты
   * [`TextBlock`](#schema-textblock) — Отображение текста, позволяя контролировать размеры шрифтов, вес и цвет.
   * [`Image`](#schema-image) — Отображает изображение.
   * [`Media`](#schema-media) — Отображает проигрыватель мультимедиа для аудио и видео содержимого.
   * [`MediaSource`](#schema-mediasource) -Определяет источник для элемента мультимедиа

Контейнеры
   * [`Container`](#schema-container) -Containers группировать элементы друг с другом.
   * [`ColumnSet`](#schema-columnset) -Набор столбцов делит область на столбцы, что элементы располагаются side-by-side.
   * [`Column`](#schema-column) — Определяет контейнер, который является частью набора столбцов.
   * [`FactSet`](#schema-factset) -Элемент FactSet выводит ряд факты (т. е. пары имя/значение) в табличной форме.
   * [`Fact`](#schema-fact) : Описаны факта в FactSet как пара ключ/значение.
   * [`ImageSet`](#schema-imageset) -ImageSet отображает коллекцию изображений, аналогичную коллекции.

Действия
   * [`Action.OpenUrl`](#schema-action.openurl) -При вызове Показать заданный URL-адрес либо путем запуска его в внешних веб-браузер или отображение на месте с помощью внедренных веб-браузер.
   * [`Action.Submit`](#schema-action.submit) — Собирает поля ввода, выполняет слияние с полем необязательные данные и отправляет событие в клиент. Он зависит от клиента, чтобы определить, как обрабатывается эти данные. Пример: С помощью программы-роботы BotFramework клиент отправит действие через среду обмена сообщениями бота.
   * [`Action.ShowCard`](#schema-action.showcard) -Определяет AdaptiveCard, который видит пользователь при нажатии кнопки или ссылки.

Ввод данных
   * [`Input.Text`](#schema-input.text) -Позволяет пользователю вводить текст.
   * [`Input.Number`](#schema-input.number) -Позволяет пользователю ввести номер.
   * [`Input.Date`](#schema-input.date) -Позволяет пользователю выбрать дату.
   * [`Input.Time`](#schema-input.time) -Позволяет пользователю выбрать время.
   * [`Input.Toggle`](#schema-input.toggle) -Позволяет пользователю выбрать один из двух вариантов.
   * [`Input.ChoiceSet`](#schema-input.choiceset) -Позволяет пользователю вводить выбор.
   * [`Input.Choice`](#schema-input.choice) : Описаны выбором для использования в ChoiceSet.
