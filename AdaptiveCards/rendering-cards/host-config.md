# <a name="what-is-hostconfig"></a>Что такое HostConfig?
`HostConfig` — это **кроссплатформенный объект конфигурации**, указывающий, как отрисовщик адаптивных карточек создает пользовательский интерфейс.

Это позволяет отрисовщикам на разных платформах и устройствах использовать свойства, которые независимы от платформы. Кроме того, это позволяет создавать наборы инструментов, которые дают представление о внешнем виде и функциях карточки в конкретной среде.

Ознакомьтесь с примером [HostConfig.JSON](https://github.com/Microsoft/AdaptiveCards/blob/master/samples/HostConfig/sample.json), чтобы изучить его содержимое.

---

   * [`AdaptiveCardConfig`](#schema-adaptivecardconfig): параметры верхнего уровня для `AdaptiveCards`.
   * [`ActionsConfig`](#schema-actionsconfig): параметры для объектов `Action`.
   * [`ContainerStylesConfig`](#schema-containerstylesconfig): управляет стилем контейнера по умолчанию и контейнера выделения.
   * [`FactSetConfig`](#schema-factsetconfig): управляет отображением объектов `FactSet`.
   * [`FontSizesConfig`](#schema-fontsizesconfig): управляет метриками размера шрифта для различных стилей текста.
   * [`FontWeightsConfig`](#schema-fontweightsconfig): управляет метриками насыщенности шрифта.
   * [`ForegroundColorsConfig`](#schema-foregroundcolorsconfig): управляет различными цветами шрифтов.
   * [`ImageSetConfig`](#schema-imagesetconfig): управляет отображением объектов `ImageSet`.
   * [`ImageSizesConfig`](#schema-imagesizesconfig): управляет размером объектов `Image`.
   * [`MediaConfig`](#schema-mediaconfig): управляет отображением и поведением элементов `Media`.
   * [`SeparatorConfig`](#schema-separatorconfig): управляет отображением разделителей.
   * [`ShowCardConfig`](#schema-showcardconfig): управляет поведением и стилем объектов `Action.ShowCard`.
   * [`SpacingsConfig`](#schema-spacingsconfig): управляет расположением элементов.
   * [`TextBlockConfig`](#schema-textblockconfig): параметры, управляющие отображением текста.
