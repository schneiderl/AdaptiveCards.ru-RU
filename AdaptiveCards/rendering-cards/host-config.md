# <a name="what-is-hostconfig"></a>Что такое HostConfig?
`HostConfig` — **объект конфигурации кросс платформенные** , указывающий, что создание адаптивной модуля подготовки отчетов карты пользовательского интерфейса.

Благодаря этому свойства, которые зависят от платформы для использования несколькими модулями подготовки отчетов на разных платформах и устройствах. Он также позволяет набор инструментов, чтобы создать, который дает представление о внешнего вида и поведения в карточке, будет иметь для данной среды.

Пример см. в разделе [HostConfig.json](https://github.com/Microsoft/AdaptiveCards/blob/master/samples/HostConfig/sample.json) чтобы почувствовать за их содержимое.
